# Blood Donation Management System

## Description
This project is a Blood Donation Management System developed using Python (Tkinter) and SQLite. It helps in managing donor and patient information and makes it easier to find suitable blood donors during emergencies.

## Features
- User registration and login
- Password security using hashing
- Add and view donors
- Add and view acceptors
- Search nearby donors based on location
- Forgot password option

## Technologies Used
- Python
- Tkinter
- SQLite

## How to Run
1. Install Python
2. Install required library:
   pip install pillow
3. Run the program:
   python main.py

## Purpose
The main goal of this project is to simplify blood donation management and help people quickly find donors when needed.
